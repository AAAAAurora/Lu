一种基于不确定性感知与执行后残差校正的视觉—语言导航多候选未来状态推理方法 Uncertainty-Aware Multi-Candidate Future State Reasoning for Vision-and-Language Navigation with Post-Execution Residual Correction

方向：视觉语言导航、具身智能、未来状态推理 Area: Vision-and-language navigation, embodied intelligence, future state reasoning

技术概述 Technical Overview

该方法面向视觉语言导航任务，不再只对最高分候选节点进行单一路径前瞻，而是对多个候选节点并行预测执行后的未来视觉状态，并为每个预测结果估计不确定性。系统随后结合语言匹配、拓扑一致性、预测可靠性和历史残差对候选动作重评分。 This method targets vision-and-language navigation. Instead of reasoning only over the top-scored candidate node, it predicts future visual states for multiple candidates in parallel and estimates the uncertainty of each prediction. Candidate actions are then rescored by combining language alignment, topological consistency, prediction reliability, and historical residuals.

在动作执行后，系统将真实后继观测与执行前预测状态进行比较，把预测—观测残差回写至拓扑图和后续决策流程，从而形成“动作前预测、动作后校验、异常时重规划”的闭环。 After action execution, the system compares the real successor observation with the predicted future state and feeds the prediction-observation residual back into the topological graph and subsequent decision process, forming a closed loop of pre-action prediction, post-action verification, and replanning under abnormal predictions.

方法流程 Workflow

  • 构建视觉观测、自然语言指令和历史导航状态对应的拓扑图表示。 Build a topological representation from visual observations, natural-language instructions, and navigation history.
  • 从候选节点中筛选 Top-K 节点，为每个候选节点构建未来状态预测条件。 Select Top-K candidate nodes and construct future-state prediction conditions for each candidate.
  • 并行生成多候选未来视觉状态，同时估计预测不确定性。 Generate future visual states for multiple candidates while estimating prediction uncertainty.
  • 利用不确定性感知门控控制未来状态对节点表示和动作评分的反馈强度。 Use uncertainty-aware gating to control how strongly future states affect node representations and action scoring.
  • 动作执行后计算预测—观测残差，并根据残差修正候选置信度或触发自适应重规划。 After execution, compute prediction-observation residuals and use them to adjust candidate confidence or trigger adaptive replanning.

创新点 Key Innovations

  • 多候选并行未来状态推理，降低单一候选误选带来的级联错误。 Parallel future-state reasoning over multiple candidates reduces cascading errors caused by a wrong single-candidate choice.
  • 不确定性感知反馈门控，使可信预测被增强、不可信预测被抑制。 Uncertainty-aware feedback gating strengthens reliable predictions and suppresses unreliable ones.
  • 执行后残差校正机制，使系统能够根据真实观测在线校验并纠偏。 Post-execution residual correction allows online verification and correction using real observations.
  • 异常预测触发自适应重规划，提升遮挡、转角和长路径场景下的稳定性。 Adaptive replanning under abnormal predictions improves robustness in occluded, turning, and long-horizon navigation scenarios.
